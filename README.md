# Exact-Ti Security Platform — Dashboard Plugins

> Plugins da interface web da Exact-Ti Security Platform.

## Sobre

Este repositório contém os plugins do dashboard da **Exact-Ti Security Platform**:

| Plugin | ID | Descrição |
|---|---|---|
| Main | `exactti` | Plugin principal — alertas, agentes, relatórios |
| Core | `exactti-core` | Serviços compartilhados e utilitários |

## Rotas

- `/app/exactti` — Interface principal
- `/app/exactti#/overview` — Visão geral
- `/app/exactti#/agents` — Gerenciamento de agentes

## Índices

| Índice | Descrição |
|---|---|
| `exact-ti-alerts-*` | Alertas de segurança |
| `exact-ti-monitoring-*` | Monitoramento de agentes |
| `exact-ti-statistics-*` | Estatísticas da plataforma |
| `.exactti` | Configurações internas |
